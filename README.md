# NYC-Rideshare-Analysis-Part-2
A year-over-year analysis of 400M+ NYC rideshare trips examining the impact of the $1.50 congestion fee implemented on January 5, 2025. Using BigQuery and Looker Studio, this analysis compares full year 2024 (pre-fee) vs 2025 (post-fee) across trip volume, net revenue, and driver earnings: citywide and at the borough level.
